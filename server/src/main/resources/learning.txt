List<Integer> liczby = Arrays.asList(42, 12, 89, 7, 55);

Optional<Integer> min = liczby.stream()
                              .min(Integer::compare);

// Wyciąganie wartości z Optionala:
if (min.isPresent()) {
    System.out.println("Najmniejsza liczba to: " + min.get()); // Wynik: 7
}

// Lub krócej (jeśli brak, zwróć 0):
Integer wynik = min.orElse(0);

String result = list.stream()

    .findAny()                   // Zwraca Optional<String>
    .orElse("nnn");              // Jeśli pusty, daj "nnn"
    
    
    
    
    
    
    
    
    Integer l = list.stream()
    .filter(t -> t % 7 == 0)
    .findFirst()         // Tu mamy Optional<Integer>
    .map(t -> t * 10)    // Mapujemy środek Optionala (tylko jeśli coś w nim jest)
    .orElse(0);          // Na samym końcu decydujemy, co jeśli pusto
    filter przed find: Zawsze najpierw filtrujesz strumień, a potem "chwytasz" wynik przez findFirst() lub findAny().
Mapowanie Optionala: Metoda .map() na Optionalu jest genialna, bo wykona się tylko wtedy, gdy element został znaleziony. Jeśli filter nic nie znajdzie, map zostanie bezpiecznie pominięty.
get() to ostateczność: Unikaj .get(), dopóki nie sprawdzisz .isPresent(). Lepiej używać orElse, orElseGet lub map.
